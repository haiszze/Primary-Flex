Customized MultiSelect Configuration

Adaptive PrimeNG MultiSelect dropdown structure and styling for mobile and desktop using PlatformUtil.mobile() as the branch condition.

Overlay Target & Cancel Button Injection root

Overrides the append target signal and injects a sticky Cancel footer on mobile.

Mobile: appends the overlay to the document body to escape parent overflow and z-index constraints, then injects a sticky Cancel button at the bottom of the overlay. Clicking Cancel hides the panel and clears the current value.

Desktop: appends the overlay inline next to the trigger element within its normal stacking context.

Overlay Backdrop, Positioning & Bottom Sheet Shape overlay

Combines backdrop styling, overlay positioning, and panel shape into a single key when the overlay is appended to the body.

Mobile: full-screen semi-transparent black backdrop covering the entire viewport, panel anchored to the bottom and centered horizontally at 60vw (full-width on small screens), capped at 80vh, flex-column layout, and bottom border radius removed for a native bottom-sheet feel.

Desktop: no backdrop, constrained to the full width of its container with standard overflow handling.

Header Padding header

Static class applied to the panel header regardless of platform. Reduces the default horizontal padding to align the search input and select-all checkbox consistently with the option list below.

Scrollable List Area listContainer

Mobile: expands to fill all remaining vertical space inside the bottom sheet, allowing the list to scroll independently while the Cancel footer stays pinned.

Desktop: removes default horizontal padding on both the container and the inner list element for edge-to-edge option alignment.

Option Label Text Wrapping option

Static classes applied to all option rows regardless of platform. Targets the inner content node via child selectors to override PrimeNG's default truncation, allowing long labels to wrap across multiple lines, break at word boundaries, hyphenate automatically, and occupy the full available width.
