Customized Select Configuration

Adaptive PrimeNG Select dropdown structure and styling for mobile and desktop using PlatformUtil.mobile() as the branch condition.

Overlay Target & Cancel Button Injection root

Overrides the append target signal and injects a sticky Cancel footer on mobile.

Mobile: appends the overlay to the document body to escape parent overflow and z-index constraints, then injects a sticky Cancel button at the bottom of the overlay.

Desktop: appends the overlay inline next to the trigger element within its normal stacking context.

Overlay Backdrop & Positioning pcOverlay › root

Mobile: full-screen semi-transparent black backdrop covering the entire viewport.

Desktop: no backdrop, constrained to the full width of its container.

Bottom Sheet vs Dropdown Shape pcOverlay › content

Mobile: anchors the panel to the bottom, centers it horizontally, caps height at 80vh, flex-column layout, and removes the bottom border radius.

Desktop: preserves PrimeNG's positioning, full height with hidden overflow for clean scrolling.

Scrollable List Area listContainer

Mobile: expands to fill all remaining vertical space inside the bottom sheet.

Desktop: removes default horizontal padding for edge-to-edge option alignment.

Option Label Text Wrapping optionLabel

Static classes applied to all option labels regardless of platform. Overrides PrimeNG's default truncation so long labels wrap across multiple lines, break at word boundaries, and hyphenate automatically.
